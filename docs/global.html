 Permissions required by the user to use the command.

 Options for throttling usages of the command.

 true Whether or not the default command handling should be used. If false, then only patterns will trigger the command.
args Array.<ArgumentInfo> <optional>
 Arguments for the command.
argsPromptLimit number <optional>
 Infinity Maximum number of times to prompt a user for a single argument. Only applicable if `args` is specified.
argsType string <optional>
 single One of 'single' or 'multiple'. Only applicable if `args` is not specified. When 'single', the entire argument string will be passed to run as one argument. When 'multiple', it will be passed as multiple arguments.
argsCount number <optional>
 0 The number of arguments to parse from the command string. Only applicable when argsType is 'multiple'. If nonzero, it should be at least 2. argsSingleQuotes boolean <optional>
 true Whether or not single quotes should be allowed to box-in arguments in the command string.
patterns Array.<RegExp> <optional>
 Patterns to use for triggering the command
guarded boolean <optional>
 false Whether the command should be protected from disabling
